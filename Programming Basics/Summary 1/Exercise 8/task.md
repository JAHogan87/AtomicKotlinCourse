## Summary 1 (#8)

Reverse the digits in a decimal number using a `while` loop. An auxiliary
`var worker` will help you iteratively produce the `result`:

| Worker     | Result |
|------------|--------|
| 1234       | 0      |
| 123        | 4      |
| 12         | 43     |
| 1          | 432    |
| 0          | 4321   |
